# Lazy Pizza

A React application that allows users to browse a menu of pizzas, put it into a cart, and place an order.

## Features

- App doesn`t require login, user can input their name before placing an order
- User can order one or more pizzas from a menu
- Menu loading from an API
- User can add pizzas to a cart and change its quantity
- User's address is pre-filled based on their GPS location (need user permission)
- User can mark order as a priority for a small price
- User can update order to make it a priority
- Each order has a unique ID that can be used to search an order and see order details later

## Technologies used

- React v18+
- React Router v6.4+
- Redux Toolkit
- TailwindCSS
